1. What is DOM and what is the purpose of DOM?

DOM is Document Object Model. It can change the website interface as like as we want. The purpose of DOM is that DOM helps us to create our website more dynamic and interactive using styles and other attributes.

2. How will you select HTML elements using DOM? Name the DOM methods

It's depending on us what we want to do. For example:
If we want to get a specific ID we can use 'document. getElementById():'
To get all elements with a specific tag name we can use 'document. getElementsByTagName();'
If we want to get all elements with a specific class Name we can use 'document.getElementsByClassName ():'
if we want to get the first element that matches the CSS selector we can use 'document.querySelector();'
On the other hand, If we want to get all elements that match the CSS selector we can use 'document.querySelectorAll();'

3. What is event bubble?

Event bubble will select the target elements. This event first selects the target element, and then it's go to the parent element, and it will continue until it reaches the top level of document objects
